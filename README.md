## ⚙️ Instalação
```
# Abra um terminal e copie este repositório com o comando
$ git clone https://github.com/MikeFernando/spiderman.git
```


```
# Acesse a pasta da aplicação
$ cd spiderman

# Instale as dependências
$ npm

# ou instale usando yarn
$ yarn

# Inicie a aplicação
$ npm start

```
